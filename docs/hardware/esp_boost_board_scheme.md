# ESP_Boost Board Scheme

Источник: Google Docs `ESP_Boost`, вкладка `Описание платы`.

## Блок питания и общие земли

Все земли (`GND`) модулей должны быть надежно соединены вместе.

- Авто `+12V` (после замка и предохранителя) -> вход `IN+` на `LM2596HVS`, плюс на `MAC-соленоид`, пин `6 (VS)` на `L9637D`
- Авто `GND` -> вход `IN-` на `LM2596HVS`, общая земля схемы
- `LM2596HVS OUT+` (настроен строго на `5.0V`) -> `VIN` (или `5V`) пин `ESP32`, `VDD` на `ADS1115`, `HV` на `LLC`, пин `3 (VCC)` на `L9637D`
- `ESP32 3.3V` (пин `3V3`) -> `LV` на `LLC`, подтягивающие резисторы для транзисторов (обороты/скорость)

## 2. Аналоговая часть (`ADS1115`)

Защита ставится до входа в АЦП. `ADS1115` питается от `5V`, чтобы иметь нужный диапазон измерения.

- Сигнал буста -> резистор `1 кОм` -> пин `A0 (ADS1115)`
- Защита на пине `A0`: катод стабилитрона `1N4733A 5.1V` к `A0`, анод к `GND`; конденсатор `0.1 мкФ` между `A0` и `GND`
- Сигнал дросселя -> резистор `10 кОм` -> пин `A1 (ADS1115)`
- Защита на пине `A1`: катод стабилитрона `1N4733A 5.1V` к `A1`, анод к `GND`; конденсатор `0.1 мкФ` между `A1` и `GND`

## 3. Цифровая шина `I2C` (`ADS1115` <-> `ESP32`)

Сигналы `I2C` 5-вольтового АЦП согласовываются с 3.3-вольтовой `ESP32` через конвертер (`LLC`).

- `ADS1115 SCL` -> `HV2 (LLC)` -> `LV2 (LLC)` -> пин `22 (ESP32)`
- `ADS1115 SDA` -> `HV1 (LLC)` -> `LV1 (LLC)` -> пин `23 (ESP32)`

## 4. Дискретные входы (обороты и скорость)

Используются `2N3904` как инвертирующие ключи (`Level Shifter`) для перевода автомобильных `12V` импульсов в безопасные `3.3V`.

### Сигнал оборотов (тахометр)

- Сигнал оборотов -> резистор `47 кОм` -> база `2N3904 (№1)`
- От базы транзистора -> стягивающий резистор `10 кОм` -> `GND`
- Эмиттер транзистора -> `GND`
- Коллектор транзистора -> пин `18 (ESP32)` и подтягивающий резистор `4.7 кОм` на `3.3V`

### Сигнал скорости

- Сигнал скорости -> резистор `47 кОм` -> база `2N3904 (№2)`
- От базы транзистора -> стягивающий резистор `10 кОм` -> `GND`
- Эмиттер транзистора -> `GND`
- Коллектор транзистора -> пин `19 (ESP32)` и подтягивающий резистор `4.7 кОм` на `3.3V`

## 5. Силовой выход (`MAC` соленоид через `LR7843`)

Обычно такие модули коммутируют "минус" (`Low-side switch`).

- `+12V` (после предохранителя) -> напрямую к плюсовому проводу соленоида
- Минусовой провод соленоида -> "средний пин" (`Drain`) модуля `LR7843`
- `GND` модуля `LR7843` -> общая земля
- Управляющий пин (`Signal/PWM`) модуля `LR7843` -> пин `25 (ESP32)`
- Обязателен выпрямительный диод `1N4007` параллельно контактам соленоида: катод к `+12V`, анод к минусовому проводу, идущему на `Drain`

## 6. Чтение `K-Line` (`L9637D`)

`L9637D` работает на `5V` логике, а `ESP32` на `3.3V`. Для согласования в исходной схеме используются оставшиеся 2 канала конвертера уровней (`LLC`). Для `ESP32` указан `Serial2` (пины `16` и `17`).

- Пин `1 (RX)` (`выход 5V`) -> `HV3 (LLC)` -> `LV3 (LLC)` -> пин `16 / RX2 (ESP32)`
- Пин `2 (TX)` (`вход 5V`) -> `HV4 (LLC)` <- `LV4 (LLC)` <- пин `17 / TX2 (ESP32)`
- Пин `3 (VCC)` -> `5V` (от `LM2596`)
- Пин `4 (GND)` -> общая земля
- Пин `5 (GND)` -> общая земля
- Пин `6 (VS)` -> `+12V` (от бортсети авто)
- Пин `7 (K)` -> провод `K-Line` к колодке диагностики ЭБУ `OBD2`
- Пин `8` -> не подключается (`NC`)
