# Firmware

В этой папке лежит прошивка ESP32 для проекта `YRV_Boostcontrol`.

## Основной файл

- `boost_controller_refined/boost_controller_refined.ino`

## Что делает прошивка

- управляет соленоидом бустконтроля
- читает MAP/TPS через ADS1115
- считает RPM и скорость
- отдает телеметрию по BLE
- принимает команды настройки от Android-приложения
- сохраняет параметры в NVS
- поддерживает OTA-обновление через Wi-Fi

## Перед прошивкой

Проверь:

- правильную плату ESP32
- установленные библиотеки
- соответствие пинов реальному подключению
- корректность калибровок датчиков после первого запуска
